# Kablosuz-aglar-projesi-kablosuz alarm sistemi 
Bu kod, bir ESP32 kartı (veya benzeri bir Wi-Fi özellikli mikrodenetleyici) için yazılmış, hareket algılayıp MQTT protokolü üzerinden bir sunucuya bilgi gönderen bir programdır.
Emeği Geçenler Oktay Kaya-Ahmet Can Küçük-Abdulbaki Pişirici-Muhammed Emin Yazar-Ceyda Uzun-İrem Utuş.
Doktor Öğretim Üyesi Hasan Serdar'ın kablosuz ağlar dersi için yapılmıştır.
